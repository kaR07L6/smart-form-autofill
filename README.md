# Smart Form Autofill

フォーム自動入力Chrome拡張機能

## セキュリティ機能

- **AES-256-GCM暗号化**: すべてのデータを暗号化
- **ローカル保存**: データはすべて端末内に保存され、外部送信なし
- **ドメイン別学習**: サイトごとに異なるデータを安全に管理
- **自動クリーンアップ**: 30日以上使用されないデータを自動削除

## 機能

- フィールドタイプの自動検出（メール、電話、住所等）
- 入力履歴の学習と自動補完
- サイトごとのカスタム学習
- リアルタイムサジェスト
- 動的フォームにも対応

## インストール方法

1. このリポジトリをクローン
2. Chromeで `chrome://extensions/` を開く
3. 「デベロッパーモード」を有効化
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## 使い方

1. 拡張機能をインストール
2. フォームに情報を入力して送信
3. 次回から自動で補完候補が表示されます

## プライバシー

- データは暗号化されてローカルに保存
- 外部サーバーへの送信は一切なし
- ユーザーが完全にコントロール可能
